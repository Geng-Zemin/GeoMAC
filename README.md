GeoMAC
Official repository for the paper:

"GeoMAC: Robust Multimodal Image Matching via Virtual Affine Manifold and Geometric-Physical Constraints"

📂 Repository Contents
GeoMAC_Matlab.exe: Executable Program. The standalone Windows executable used to perform multimodal image matching and quantitative evaluation as described in the paper.

Data.7z: Test Datasets. A compressed package containing the 7 typical heterogeneous datasets used for validation (including Medical, Optical-SAR, Infrared, Depth, Map, Heterogeneous Optical, and Nighttime imagery).

🚀 Quick Start
Download: Download both AHMIRRIFT.exe and Data.7z.

Extract: Unzip Data.7z to access the test image pairs.

Run: Launch GeoMAC_Matlab.exe on Windows. Select a reference image and a sensed image from the data folder to start the matching process.
